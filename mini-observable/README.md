# MiniObservable — 响应式编程

从零实现 RxJS 风格 Observable 与操作符（~170 行）。

- **Observable** — 推送式数据流，subscribe / pipe
- **操作符** — `map` / `filter` / `take` / `debounceTime` / `merge`
- **创建函数** — `fromEvent` / `interval` / `of`
- **Subject** — 多播 Observable
- **Subscription** — 可取消订阅，自动清理
