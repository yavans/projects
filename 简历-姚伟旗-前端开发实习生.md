## 核心项目

### MiniReact — 自研 Virtual DOM & Hooks 框架 | ~350 行

从零实现 React 核心机制，涵盖 Virtual DOM diff、Fiber 调度与 Hooks 系统。

- 实现 **同层 diff 算法**，通过 type 比对生成 UPDATE / PLACEMENT / DELETION 操作指令
- 引入 **Fiber 架构**，基于 `requestIdleCallback` 实现可中断的分片渲染，避免主线程阻塞
- 实现 **useState**（状态队列 + 批量更新）与 **useEffect**（依赖比较 + cleanup 回调）
- 支持 Function Component + 合成事件系统

### MyPromise — Promise/A+ 规范完整实现 | ~230 行

严格遵循 [Promise/A+ 规范](https://promisesaplus.com/)，完整实现 Promise 全部特性。

- 实现**三态状态机**（pending → fulfilled / rejected）与异步回调队列
- 正确实现 **Promise Resolution Procedure**（§2.3）：thenable 递归解包、循环引用检测、多次调用保护
- 扩展实现全部静态方法（`all` / `race` / `allSettled` / `any` / `resolve` / `reject`）
- 扩展实现全部实例方法（`then` / `catch` / `finally`）

### FullStack Blog — Express + JWT 全栈博客

前后端分离的全栈博客平台，后端 RESTful API，前端 Vanilla JS SPA。

- **后端**：Express 5 + JWT（bcryptjs）认证，JSON 文件持久化，模块化路由设计
- **前端**：原生 JS 实现 Hash Router SPA + 暗色主题 + 响应式布局
- 实现用户注册/登录、文章 CRUD（Markdown 格式）、评论系统、标签分类、分页查询、管理后台

### CodePlay — 在线代码游乐场（类 CodePen）

三栏实时代码编辑器，支持 HTML / CSS / JS 编辑与即时预览。

- 实现**可拖拽三栏布局**，Tab 缩进支持，localStorage 自动保存
- iframe 沙箱隔离预览区，**250ms 防抖**自动刷新，避免高频 DOM 操作
- 内置 Console 面板，重写 `console.log/error/warn` 捕获输出 + 运行时错误
- 支持 URL hash 编码分享 + 一键导出 HTML 文件

### Algorithm Visualizer — 算法可视化演示

Canvas 动画展示排序算法与路径搜索算法的执行过程。

- 实现 **5 种排序算法**（Quick / Merge / Heap / Bubble / Insertion）的可视化，实时显示比较/交换次数与颜色标注
- 实现 **4 种路径搜索算法**（A\* / Dijkstra / BFS / DFS），支持鼠标绘制障碍物与拖动起终点
- 通过可调节速度控制，直观对比不同算法的搜索策略与效率差异

### Virtual Scroller — 高性能虚拟滚动

仅渲染可视区域 DOM，支持 **10 万+行数据流畅滚动**。

- 基于 `requestAnimationFrame` 合并高频 scroll 事件，避免卡顿
- 实现 DOM 回收复用机制（可见行 + 8 行缓冲区），滚动时更新数据内容而非重建节点
- 内建性能仪表盘：实时 FPS / DOM 节点数 / 渲染耗时监控
