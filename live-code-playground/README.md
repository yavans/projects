# CodePlay — Live Code Playground

在线代码游乐场，类似 CodePen。HTML/CSS/JS 三栏实时编辑 + 预览。

## 功能

- **三栏编辑器** — HTML / CSS / JavaScript 独立面板，可拖拽调整大小
- **实时预览** — 输入即刷新（250ms 防抖），iframe 沙箱隔离
- **Console 面板** — 捕获 `console.log/error/warn` + 运行时错误
- **5 套模板** — Hello World · Counter · Todo · Canvas 动画 · Fetch API
- **自动保存** — localStorage 持久化，刷新不丢失
- **下载** — 一键导出为独立 HTML 文件
- **分享** — 编码到 URL hash，复制链接即可分享
- **键盘快捷键** — Tab 缩进、Ctrl+S 强制刷新预览
