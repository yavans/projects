## 核心项目

### MiniVue — Proxy 响应式系统 | ~160 行

从零实现 Vue 3 风格响应式系统，深入理解依赖收集与触发更新机制。

- 基于 **Proxy + Reflect** 实现深层递归响应式代理，使用 `WeakMap<target, Map<key, Set<effect>>>` 进行依赖追踪
- 实现 **effect** 与清理机制：每次重新执行前从所有依赖集合中移除自身，防止"鬼依赖"与内存泄漏
- 实现 **computed**：惰性求值 + 脏标记缓存，仅在依赖变化且被访问时重新计算
- 实现 **watch**：通过 `traverse` 深度访问源对象以收集全部依赖，支持 getter 函数与响应式对象两种监听源

### MiniReact — Virtual DOM & Fiber 协调器 | ~350 行

从零实现 React 核心渲染引擎，涵盖 Virtual DOM diff、Fiber 架构与 Hooks 系统。

- 实现**同层 diff 算法**，通过 type 比对生成 UPDATE / PLACEMENT / DELETION 操作指令
- 引入 **Fiber 架构**，基于 `requestIdleCallback` 实现可中断的分片渲染（构建阶段 + 提交阶段）
- 实现 **useState**（per-fiber 状态队列 + 批量更新）与 **useEffect**（依赖比较 + cleanup 回调）
- 区分 Host Component 与 Function Component 的渲染路径

### MiniRedux — 状态管理库 | ~160 行

从零实现 Redux 核心：`createStore`、`combineReducers`、`applyMiddleware`。

- 实现 **dispatch 互斥锁**（`isDispatching` 标志），防止嵌套 dispatch 与派发过程中的状态读取
- 实现 **combineReducers** 引用相等性优化（`!==`），跳过未变更的 state slice 对应的 listener 通知
- 基于 **reduceRight** 实现洋葱模型中间件链组合，实现 logger（分组日志）与 thunk（异步 action）中间件

### MiniRouter — SPA 路由器 | ~140 行

从零实现 SPA 路由器，支持 Hash / History 双模式。

- 通过**正则命名捕获组**将 `/user/:id` 转换为参数提取，实现动态路由匹配
- 实现 **beforeEach 导航守卫**异步管道：返回 `false` 取消、返回字符串重定向、返回 `undefined` 放行
- 基于 `data-link` 属性实现**声明式导航拦截**，`e.target.closest('[data-link]')` 捕获事件

### MiniObservable — 响应式编程库 | ~160 行

从零实现 RxJS 风格 Observable 与操作符。

- 实现 **pipe 模式**：通过 `reduce` 组合操作符链，支持声明式数据流转换
- 实现 **操作符**：`map` / `filter` / `take` / `debounceTime` / `merge`
- 实现 **Subject**（既是 Observable 又是 Observer）与 **Subscription 树**（父级取消自动清理子级）

### MyPromise — Promise/A+ 规范完整实现 | ~230 行

严格遵循 Promise/A+ 规范，通过官方测试套件验证。

- 实现三态状态机 + 异步回调队列 + Promise Resolution Procedure（thenable 递归解包、循环引用检测）
- 扩展 `all` / `race` / `allSettled` / `any` 等全部静态与实例方法
